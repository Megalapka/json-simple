# Практика работы с json. Библиотека json-simple

## JsonParser
Реализован парсер расположенного в проекте файла response.json. Можно менять специальность в переменной special.

## FindTicketByUser
Загрузка файла json по ссылке. Поиск специальности производится в интеррактивном режиме. В консоли предлагается выбрать и ввести номер необходимой специальности.
Пока реализация на этапе - сохранить полученный json во временный файл. Данная реализация решает проблему кодировки в IntelliJ IDEA и консоли. Почему-то полученный и обработанный напрямую файл json некорректно кодирует кириллицу. Какие только настройки не перепробовала! Проблему решило использование класса FileReader. Здесь же есть функционал отслеживания талонов по таймеру.

## FindTicketByFileConfig
Загрузка файла json по ссылке, но с использованием файла настроек, который выключает в себя данные района, специальности, и хотелось бы сделать и врача. Подумать над возможностью сохранить последние выбранные настройки. Это будет улучшение для FindTicketByUser. (Не реализовано. Следующий этап по развитию проекта.)
